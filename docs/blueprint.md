# **App Name**: RecipePal

## Core Features:

- Recipe Import: Allow users to import recipes from URLs like https://crouton.app/ or manually enter their own recipes.
- Recipe Collections: Enable users to organize their recipes into custom collections.
- Meal Planning Calendar: Provide a calendar interface for creating meal plans.
- Shopping List Generation: Automatically generate shopping lists from planned meals using AI to consolidate ingredients. Use a tool to remove duplicate ingredients and suggest quantities based on recipe servings.

## Style Guidelines:

- Primary color: Light green (#A7D9A8) to evoke freshness and health.
- Background color: Cream (#F5F5DC) to provide a warm, inviting feel.
- Accent color: Soft brown (#B8860B) for highlights and calls to action, creating contrast without overwhelming the light theme.
- Body and headline font: 'PT Sans' for a modern and readable interface.
- Use simple, clean icons to represent different recipe categories and actions.
- Employ a clean and minimalist layout to ensure ease of navigation and readability.
- Incorporate subtle animations for adding recipes or completing tasks.
